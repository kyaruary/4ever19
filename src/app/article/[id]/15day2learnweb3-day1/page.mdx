# 什么是 web

万维网(World Wide Web)是一种是一个透过互联网访问的, 由许多互相链接的超文本组成的信息系统。

初始的 web 也就是我们现在所谓的 web1 核心是 HTTP(Hyper Text Transport Protocol) + HTML(Hyper Text Markup Language)

这个时候的页面也称作文档, 从 html 中的 document 标签可以发现这个特性。

时代的局限性让当时的网页只可以阅读和简单的表单操作, 它把传统的纸质形式文档和表单转化成了可以在互联网上可以传输的数据。

web1 以一种静态的方式展示在我们面前, 开发人员编写好静态的 html 页面并部署在服务器上, 用户即可以通过浏览器访问到。

对于开发人员来讲开发 web1 并不需要掌握特别多的技巧, 所见即所得的 html 和一个服务器就可以完成任务。

# 去中心化的 web3

> Web3, 全称 Web3.0, 是互联网的第三个发展阶段, 也被誉为智能化网络或语义化网络。Web3.0 通常被理解为一个更智能, 更连接, 更开放, 更安全的互联网版本。
>
> Web3.0 在很大程度上依赖于区块链技术, 这是一种分散的技术, 可以防止数据被任何中央实体控制。这种技术使得网络变得更加透明, 私密性得到提高, 使得数据的所有权返回给用户。
>
> Web3.0 的重要组成部分是智能合约和去中心化应用程序(DApps)。智能合约是在区块链上运行的自动执行程序, 无需任何人工干预即可进行交易和议定书操作。DApps 是建立在开放性协议之上, 不属于任何个人或组织, 而且一旦发布在区块链网络上, 甚至开发者也不能做出任何修改。
>
> 总体上说, Web3.0 为创建更公平、开放的虚拟空间, 提供了新的可能性, 能够增强人和机器之间的互动和理解, 实现更自动化和个性化的网络体验。
